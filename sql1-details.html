Project information

  • Category: MySQL
  • Project Purpose: Library Database System
  • Project date: September 2023

Description

In order to pass in the Database Management module, students in APU are required to complete the DBMS group assignment. The assignment requested us to build a database that record all the data in library. Furthermore, we need to design a Entity Relationship Diagram to link all tables together to build up a management system. During the group assignment period, I had learned how to create a database, how to manage database, how to design database structure, and how to code database query.
